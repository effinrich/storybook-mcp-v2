## Overview

Sync a single component's story/test/docs

## Parameters

<ParamField path="componentPath" type="string" required>
  Path to the component file
</ParamField>

<ParamField path="generateStories" type="boolean" default="true">
  Generate story file
</ParamField>

<ParamField path="generateTests" type="boolean" default="true">
  Generate test file (Pro only)
</ParamField>

<ParamField path="generateDocs" type="boolean" default="true">
  Generate MDX docs (Pro only)
</ParamField>

<ParamField path="dryRun" type="boolean" default="false">
  Preview without writing to disk
</ParamField>
